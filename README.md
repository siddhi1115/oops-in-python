# oops-in-python
Hands-on implementation of Object-Oriented Programming in Python with real-world examples like bank system, shopping cart, and inheritance models.

## 💡 Programs Included
1. Factorial using Function
2. Prime Number Check
3. Bank Account System (Deposit, Withdraw, Balance)
4. Movie Hit/Flop Checker
5. Shopping Cart System
6. University System (Hybrid Inheritance)
7. Employee & Manager (Method Overriding)
8. Exception Handling Examples
9. File Handling with Custom Errors

## 🚀 Purpose
This repository is created to understand and practice core OOP concepts in Python through real-world examples.
